### 1. Register
curl -X POST http://localhost:3000/api/auth/register -H "Content-Type: application/json" -d "{\"email\":\"newuser@example.com\",\"password\":\"password123\",\"first_name\":\"Jean\",\"last_name\":\"Dupont\",\"birthdate\":\"1990-01-01\"}"

### 2. Login
curl -X POST http://localhost:3000/api/auth/login -H "Content-Type: application/json" -d "{\"email\":\"test@example.com\",\"password\":\"password123\"}"

### 3. Get Current User (remplacer YOUR_ACCESS_TOKEN)
curl -X GET http://localhost:3000/api/auth/me -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

### 4. Update User (remplacer YOUR_ACCESS_TOKEN)
curl -X PUT http://localhost:3000/api/auth/update -H "Authorization: Bearer YOUR_ACCESS_TOKEN" -H "Content-Type: application/json" -d "{\"first_name\":\"Jean Updated\"}"

### 5. Delete User (remplacer YOUR_ACCESS_TOKEN)
curl -X DELETE http://localhost:3000/api/auth/delete -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

### 6. Get all restaurants
curl -X GET http://localhost:3000/api/restaurants

### 7. Create restaurant
curl -X POST http://localhost:3000/api/restaurants -H "Content-Type: application/json" -d "{\"owner_id\":\"uuid-user-id\",\"name\":\"Restaurant Test\",\"email\":\"resto@test.com\",\"phone\":\"0123456789\"}"

### 8. Get restaurant by ID
curl -X GET http://localhost:3000/api/restaurants/1

### 9. Update restaurant
curl -X PUT http://localhost:3000/api/restaurants/1 -H "Content-Type: application/json" -d "{\"name\":\"Restaurant Updated\"}"

### 10. Delete restaurant
curl -X DELETE http://localhost:3000/api/restaurants/1

### 11. Get all dishes
curl -X GET http://localhost:3000/api/dishes

### 21. Create order item
curl -X POST http://localhost:3000/api/order-items -H "Content-Type: application/json" -d "{\"order_id\":1,\"dish_id\":1,\"quantity\":2,\"price\":15.99,\"user_id\":\"uuid-user\",\"person_number\":1}"

### 22. Create rating
curl -X POST http://localhost:3000/api/ratings -H "Content-Type: application/json" -d "{\"user_id\":\"uuid-user\",\"restaurant_id\":1,\"rating\":5,\"commentary\":\"Excellent restaurant!\"}"

### 23. Get rating
